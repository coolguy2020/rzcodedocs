# 版本上线公告

这一版我们上线了一个革命性的能力：

现在您可以不再上传任何代码文件，而是通过 MCP Server 让 AI 模型在本地实时访问您的项目代码，从而完全保护代码的安全性与隐私。

## 本次上线功能

### MCP Server 集成支持

我们与 rzcode.vip 深度集成，提供完整的 MCP Server 支持，让您的开发工具（如 VS Code、IDE 等）可以直接通过 MCP 协议与我们的服务通信。

**核心优势：**

- ✅ **代码不上传**：代码始终留在您的本地机器，无任何传输风险
- ✅ **实时分析**：MCP Server 实时访问您的最新代码，分析始终基于最新状态
- ✅ **隐私保护**：完全符合企业隐私政策与合规要求
- ✅ **安全透明**：所有通信基于开放标准 MCP 协议，可完全审查与控制
